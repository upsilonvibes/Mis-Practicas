# The Neon Revolution: The Rise of Synthpop

A 3-page informational website exploring the electronic music revolution of the 1980s. This project focuses on semantic HTML5 structure, multi-page navigation, and data presentation.

## 🚀 Project Overview
This website provides a deep dive into the synthpop genre, the legendary Roland TR-808 drum machine, and the chart-topping albums that defined a decade.

### Key Features
* **Multi-Page Navigation:** Consistent `<nav>` across all pages for a seamless user experience.
* **Semantic HTML:** Built using proper HTML5 tags (`<header>`, `<main>`, `<section>`, `<article>`, `<figure>`).
* **Data Presentation:** Includes a detailed technical table for instrument specifications.
* **Responsive Media:** Uses `<figure>` and `<figcaption>` for structured image presentation.
