# 🚀 Mis-Practicas (My Full-Stack Sandbox)

Welcome to my personal laboratory! This repository is a dedicated space where I track my hands-on experiments, capstone concepts, and transitions from static layouts to full-stack dynamic applications. It documents my practical journey through Advanced JavaScript, Node.js, and React.

---

## 📂 Repository Structure

The sandbox is organized into progressive milestones, tracking code evolution from pristine baselines to highly interactive experiments.

```text
Mis-Practicas/
├── 01-Lab-Rat-Original/   # Pristine HTML/CSS baseline (The Lab Rat)
├── 02-JavaScript-Lab/     # Vanilla JS Sandbox (DOM, Events, LocalStorage)
├── 03-React-Lab/        # React migration (Components, Props, Hooks)
└── 04-NodeJS-Lab/                   # Node.js backend integrations & APIs
```
## 🛠️ Tech Stack & Learning Objectives
- **Frontend Foundations:** Semantic HTML5, Responsive CSS3 (Flexbox/Grid layouts).

 - **Vanilla JavaScript (ES6+):** Advanced DOM manipulation, event bubbling, asynchronous flow, state preservation via localStorage.

 - **React ecosystem:** Component-based architectures, JSX, state management (useState, useEffect), and Single Page Application (SPA) routing.

 - **Backend & Tooling:** Node.js runtime configurations, npm dependency trees, and system Command Prompt (CMD) Git automation.
